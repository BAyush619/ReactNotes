# 📝 Google Keep Clone (React)

A feature-rich Notes App inspired by Google Keep, built using React.  
This project focuses on creating a clean UI and implementing real-world note management features.

---

## 📸 Features

### ✍️ Notes Management
- Create new notes  
- Title + Description support  
- Unique ID for each note  

### 📌 Pin System
- Pin / Unpin notes  
- Pinned notes appear at the top  
- Automatic separation of pinned and others  

### 🗑️ Delete System
- Delete notes  
- Move notes to Trash  
- Restore (if implemented)  

### 📦 Archive System
- Archive notes  
- Unarchive support  

### 🔍 Search
- Real-time search  
- Case-insensitive  
- Searches in:
  - Title  
  - Description  

### 🎨 UI Features
- Dark / Light mode  
- Grid & List view toggle  
- Sidebar toggle   

---

## 🧠 Tech Stack

- React.js  
- JavaScript (ES6+)  
- CSS
